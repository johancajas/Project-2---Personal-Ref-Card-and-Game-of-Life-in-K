Johan Cajas - Game of Life (K)
ngn/K — Mini Reference Card (concise)
-----------------------------------

Essence
- Arrays are first-class: scalars, lists, matrices are arrays.
- Verbs = functions; they are monadic (one-arg) or dyadic (two-arg).
- Composition and point-free style are common; avoid explicit loops.

Syntax basics
- Define verbs: `f:{[args] body}` or `f:{body}`.
- Apply: monad `f x` or `f[x]`; dyad `x f y` or `f[x;y]`.
- Lists: `1 2 3`; matrices: `1 2;3 4` (rows separated by `;`).
- Indexing: `a[i]` or `a[i;j]` for elements.

useful monadic verbs 
- `-x`    : negate
- `~x`    : logical NOT / complement
- `#x`    : count / shape
- `!x`    : range/enumerate
- `,x`    : enlist / wrap in list
- `'x`    : each (map) when used as adverb
- `|x`    : absolute / sign-or-reverse (dialect)
- `$x`    : format/cast to string
- `^x`    : exponent unary (dialect) / other unary op
- `@x`    : type/query (dialect)

useful dyadic verbs 
- `x+y`   : add
- `x-y`   : subtract
- `x*y`   : multiply
- `x%y`   : divide
- `x=y`   : equal (element-wise)
- `x<y`   : less-than
- `x>y`   : greater-than
- `x,y`   : join/concatenate
- `x#y`   : reshape / take
- `x!y`   : rotate/take/enumerate (dialect-dependent)

Adverbs
- `'`  (each) : apply a verb to each element (map)
- `/`  (reduce): fold a verb across a list (sum, product)
- `\` (scan) : prefix running results (partial reductions)

Common idioms
- Rotation: `rot:{[k;x] n:#x; k:k!n; (k_x),k#x}` and 2D via rank-apply
  `rot2d:{[r;c;m] rot[c]'rot[r]'m}`.
- Neighbors: sum 8 shifted boards with `rot2d` and add element-wise.
- Game of Life rule: `(n=3) | b & n=2` (birth if 3, survive if 2+and alive).
- Simulation: `sim:{[n;b] n next/b}` applies `next` n times.

Quick examples
- Define: `f:{x+1}` then `f 3` ⇒ 4
- Map: `inc:{x+1}` then `inc' 1 2 3` ⇒ 2 3 4
- Reduce sum: `+/ 1 2 3` ⇒ 6
